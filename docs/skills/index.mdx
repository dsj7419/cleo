## Using Skills

Skills are invoked via the `/skill-name` command or through the orchestrator:

```bash
/ct-epic-architect  # Plan an epic
/ct-research-agent  # Research a topic
/ct-orchestrator    # Coordinate workflow
```

<Info>
Skills use the `ct-` prefix to distinguish CLEO skills from other plugins.
</Info>
