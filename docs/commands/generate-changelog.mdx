# cleo generate-changelog

Generate Mintlify changelog from CHANGELOG.md.

<Info>**Agent Relevance**: Low - Release management tool</Info>

## Synopsis

```bash
cleo generate-changelog [LIMIT]
```

## Description

The `generate-changelog` command parses CHANGELOG.md and generates a Mintlify-compatible MDX file at `docs/changelog/overview.mdx`. It converts version headers to `<Update>` components with automatic tag detection.

## Arguments

| Argument | Description | Default |
|----------|-------------|---------|
| `LIMIT` | Maximum number of versions to include | 15 |

## Features

- Auto-detects tags from changelog content (Fix, Feature, Installer, Skills, etc.)
- Limits to 3 tags per release
- Escapes special characters to prevent MDX/JSX parse errors
- Generates RSS-compatible output

## Output Location

```
docs/changelog/overview.mdx
```

## Examples

<CodeGroup>

```bash Generate with Default Limit
cleo generate-changelog
```

```bash Generate Last 10 Versions
cleo generate-changelog 10
```

</CodeGroup>
